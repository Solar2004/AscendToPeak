# Ascend to Peak

An intelligent e-commerce platform featuring advanced neural search capabilities for health and wellness products.

## Features

- **Neural Search System**: Advanced vector-based search with graph traversal and activation spreading for intelligent product discovery
- **Knowledge Graph**: Interconnected data nodes for peptides, supplements, and health optimization
- **AI-Powered Recommendations**: Smart product suggestions based on semantic understanding
- **E-commerce Platform**: Complete marketplace for health and wellness products

## Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd AscendToPeak
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Configuration:**
    - Copy the example environment file:
      ```bash
      cp .env.example .env.local
      ```
    - Open `.env.local` and add your OpenRouter API key:
      ```env
      OPENROUTER_API_KEY=your_actual_api_key_here
      ```

4.  **Run the application:**
    ```bash
    npm run dev
    ```

## Technologies
